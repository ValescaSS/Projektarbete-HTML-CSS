Flexbox - En dimension

Flexbox är tänkt för att skapa endimensionell layout. Om man vill placera items i en riktning, antingen kolumner eller rader, då är det lättare att använda Flexbox.

hackernoon.com/the-ultimate-css-battle-grid-vs-flexbox

Flexbox egenskaper

Container

  • display:
    • flex
  • flex-direction:
    • row
    • row-reverse
    • column
    • colum-reverse
  • flex-wrap:
    • wrap
    • wrap-reverse
  • justify-content:
    • center
    • flex-start
    • flex-end
    • space-between
    • space-around
    • space-evenly
  • align-content:
    • center
    • flex-start
    • flex-end
    • space-between
    • space-around
    • space-evenly
  • align-items:
    • center
    • flex-start
    • flex-end
    • stretch

Items

  • order:
    • heltal (default 0)
  • flex-grow:
    • positiv nummer (default 0)
  • flex-shrink:
    • positiv nummer (default 1)
  • flex-basis:
    • length (default auto)
  • flex:
    • flex-grow   flex-shrink   flex-basis (default: 0 1 auto)
  • align-self:
    • flex-start
    • flex-end
    • center
    • stretch

Grid - Två dimensioner

Grid är tänkt för tvådimensionella layouter. Om man vill skapa en hel layout i två dimensioner - med både rader och kolumner - kan man använda CSS Grid.

hackernoon.com/the-ultimate-css-battle-grid-vs-flexbox
